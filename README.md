# test_workmate
Выполнены все функциональные и не функциональные требования.

файл models.py содержит класс данных. При необходимости можно удобно добавить новые классы.

файл reports.py содержит все функции и классы связанные с формированием отчета. Реализован словарь "REPORTS", с типами отчетов, на случай, если нужно будет масштабироваться и добавлять новые типы отчетов.

файл cli.py  содержит функции чтения и парсинга .csv файлов и формирования отчета.

Тесты лежат в папке tests. Покрытие составило 93% (см. скриншот pytest_coverage.png).

Скриншоты запусков с одним и двумя файлами называются "one_file.png" и "two_files.png" соответственно.


# Запуск
Для запуска программы нужно ввести следующие программы
(в requirements.txt лежат внешние библиотеки, в данном случае только tabulate):
```
pip install -r requirements.txt
python cli.py --files stats1.csv stats2.csv --report clickbait
```
Для запуска тестов
(в requirements-dev.txt лежат библиотеки pytest, pytest-cov и линтера ruff):
```
pip install -r requirements-dev.txt
pytest --cov=cli --cov=models --cov=reports
```
